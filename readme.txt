=== Mailer ===
Tags: mail, email, throttling, hostgator, dreamhost, godaddy, mass email, smtp, sendgrid, queue
Requires at least: 3.0.0
Tested up to: 3.5.2
Stable tag: trunk


Mailer queues and throttles emails from WordPress to keep the email flow within provider limits.
Every new registered user if email sending is enabled will recieve last saved email info.

== Description ==

Mailer slowly send emails to users when your website is visited. It keep the email flow within provided limits.

Mailer is compatible with WordPress and every plugin.


== Installation ==

1. Put the plugin folder into [wordpress_dir]/wp-content/plugins/
2. Go into the WordPress admin interface and activate the plugin
3. Optional: go to the options page and configure the plugin
